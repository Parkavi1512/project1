# Internship Matcher Platform

A full-stack web application that matches students with internships based on skills.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: In-memory data (for demo)

## Features
- Student skill submission
- Internship listing
- Skill-based matching logic
- REST APIs
- Modular backend structure

## Project Structure
backend/
- Express.js server
- REST API endpoints
- In-memory data handling
- Service-based architecture

frontend/
- HTML, CSS, JavaScript
- User interface for students
- API integration with backend
